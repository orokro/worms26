◆ Game_update()

void Game_update ( )

The main update method for the entire game, should be called once per frame.

The Game_update() method handles general every-frame logic, as well as calling the current game states update method as well. This is called every frame in the main-loop.

◆ gameTimers()

void gameTimers ( )

Updates all the Games main timers each frame, should be called every frame.

Updates the sudden death timer. (Frames till sudden death mode) Updates the grace timer. (Frames till worm-select grace timer is up) Updates the games main turn timer. (Frames till a turn is up)

◆ gameUpdates()
