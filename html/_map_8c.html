◆ Map_getSpawnPoint()

void Map_getSpawnPoint ( )

Sets globals, Map_lastRequestedSpawnX and Map_lastRequestedSpawnY to a valid spawn point for an item on the map.

When the map is being generated a number of valid spawn-points are also generated. This doesn't exactly "return" a point, but it selects from one of the generated ones and makes the aforementioned globals equal to the valid spawn point.

This method will also nullify the spawn point from the pool so no other items can spawn there.
