◆ Keys_update()

void Keys_update ( )

Reads the current state of the key unput, and updates our bitwise buffers.

Three longs are defined: keysDown, keysState, keysUp. These bits in these longs mirror the key states for each key matching our #defined bitmasks. keysDown only has it's bits true on the single-frame a key is down on. keysUp only has it's bits true for the single-frame a key is up on. keysState has it's bits true on every frame the key is pressed.
